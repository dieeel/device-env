# setting

## Apple ID設定

起動時に設定

## setting

### トラックパッド

#### ポイントとクリック

* タップでクリック => オン

#### その他のジェスチャ

* ページ間をスワイプ => オフ

### アクセシビリティ

#### ポインタコントロール

* トラックパッドオプション
  * ドラッグにトラックパッドを使用 => ON
  * ドラッグ方法 => 3本指のドラッグ

### キーボード

* キーのリピート => 速い
* リピート入力認識までの時間 => 短い
* 🌐キーを推して => 音声入力を開始(🌐キーを2回押す)
* 音声入力 => ON

#### キーボードショートカット

* 修飾キー
  * caps Lock と Control を入れ替え

#### 入力ソース

* 英字入力中にスペルを自動変換 => OFF
* 文頭を自動的に大文字にする => OFF
* インライン予測テキストを表示 => OFF
* スペースバーを２回押してピリオドを入力 => OFF
* スマート引用符とスマート舵手を使用 => OFF

* 日本語
  * ライブ変換 => OFF
  * タイプミスを修正 => OFF
  * 推測候補表示 => OFF
  * "￥"キーで入力をバックスラッシュに
  * 数字を全角入力OFF


### 一般

#### AirDropとHandoff

* AirDrop => すべての人

### Spotlight

* 関連コンテンツを表示 => OFF

### デスクトップとDock

* Dockを自動的に非表示 => ON
* デスクトップとステージマネージャ
  * 項目を表示 => OFF (デスクトップに／ステージマネージャ使用時に)
* 壁紙をクリックしてデスクトップ => ステージマネージャ使用時のみ
* ステージマネージャ => ON
* iPhoneウィジット => OFF

### 外観

* 外観モード => ダーク
* スクロールバーをクリックして => クリックされた部分へジャンプ

### 通知

* 画面がロックされている時 => OFF

### サウンド

* 起動時にサウンドを再生 => OFF
* ユーザーインターフェイスのサウンドエフェクトを再生 => OFF
